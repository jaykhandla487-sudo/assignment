# SDLC Phases

SDLC stands for Software Development Life Cycle.

It is a process used to develop software step by step.

## Phases of SDLC

1. Requirement Analysis  
   In this phase, developers collect information about what the client needs.

2. System Design  
   The structure and design of the software are prepared.

3. Implementation  
   Developers write the program code.

4. Testing  
   The software is tested to find and fix errors.

5. Deployment  
   The software is delivered to the users.

6. Maintenance  
   After release, the software is updated and improved when required.
